7年ぶりのSolaris x86

2005-02-22

Solaris x86で遊ぶなんて，JTRON発表のシンポジウムでScott McNearlyに発売直後のSolaris 2.6を貰って以来だから，7〜8年ぶりとなる．Gnomeが入ったら遊んでみようと思っていたので，早速Fedora Core 3のパーティションを潰してインストール．
画面解像度が1280x1024までしか設定できない(ぼくは1600x1200の液晶モニタを使っている)し，Gnomeは旧いし，Evolution 1.4に統合されているXimian Exchange Connectorから会社のExchange Server 2003に接続できないなど，Fedora Core 3と比べて気になることも多々あるけれども，やっぱりリソース周りとかは商用OSらしい丁寧さを感じる．Linux 2.6と比べてもっさりと重いのは仕方ないのかな...
サーバー性能の比較とかも興味あるけど，どっかにいいデータ落ちてないかな．いまは現場のエンジニアじゃないので，手元にベンチマークできるほどのマシンがないので入れて触って/etc以下を眺めるくらいしかできない．
